# FDS Core v3.1 Internal Research License Record

This FireWorldBench FDS Core snapshot is an internal research release derived
from locally executed FDS 6.11.1 simulations. It may be used internally for
research, model training, evaluation, and derivation. Redistribution is
restricted until an official upstream FDS/Smokeview redistribution review has
been recorded.

The event-level `license` object refers to this record. It intentionally does
not claim a public upstream licence that has not been independently verified.

Runtime provenance is retained in every Fire Event: FDS version, Smokeview
version, FD-Gen version, input hash, log reference, mesh, and random seed.
