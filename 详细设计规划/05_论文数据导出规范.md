# 05 v2 论文数据导出规范

每个发布/论文包必须包含不可变 manifest、checksums、代码/配置/数据/模型/prompt/scorer 版本和原始预测引用。表格、图和正文数字从同一机器结果源生成，不得手改。

数据统计至少同时报告：独立 Fire Event、QA、九任务、S/I/V、source domain 和 split 数量。模型结果至少报告九任务、L1/L2/L3、Overall，以及独立的 External-CFD、Experiment、Real-Image OOD、Real-Video OOD。

公开包与私有 gold/ID 映射物理分离。许可不允许再发布的来源只发布获取/校验规则、哈希和允许的派生统计。任何路径、作者信息、API 密钥、隐藏标签或旧 `LEGACY-NONCOMPARABLE` 结果进入 v2 包都应 fail closed。

当前阶段不得导出 v2 论文结果；只有 architecture/protocol artifacts 可以标记为 `2.0.0-dev`。
