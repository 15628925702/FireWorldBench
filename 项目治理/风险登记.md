# FireWorldBench v2 风险登记

| ID | 风险 | 概率/影响 | 触发器 | 缓解与验收 |
|---|---|---|---|---|
| V2-R01 | 旧 T1/T2/T3 结果被误报为 v2 | 高/高 | 新文档或论文引用旧 artifact 分数 | `LEGACY-NONCOMPARABLE` 隔离；v2 发布扫描旧 task/config/run ID |
| V2-R02 | 相邻切片虚增独立 event 数 | 高/高 | event ID 在切片后生成；event/QA 只报一个数量 | event 在切片前生成；同时报告 event 和 QA；event_group 审计 |
| V2-R03 | event_group 或反事实 family 跨 split | 中/高 | 同 group 多 partition | split-before-derivation validator；允许重叠 0 |
| V2-R04 | 文件名、路径、EXIF 或配置泄漏标签 | 高/高 | 原始火源/HRR/通风编码进入公开输入 | 随机 ID、去元数据、标签关键词与可预测性审计 |
| V2-R05 | 外部来源被补造不存在的物理标签 | 高/高 | D-Fire/Fire360 出现机制/趋势/HRR；PolyU 出现无依据 CO/soot | source-task-track allowlist 和逐字段 origin validator |
| V2-R06 | 困难负例由视觉捷径可解 | 中/高 | 候选背景/亮度/视角/烟量或长度不同 | 同几何/视角/时长匹配；appearance-confounded 子集；人工抽检 >=90% |
| V2-R07 | 时间位置成为固定风险捷径 | 中/高 | 标签与窗口时长/绝对时间高度相关 | 时间匹配与 shortcut report；按窗口分层统计 |
| V2-R08 | FDS 网格不稳定或成本超过资源 | 高/高 | pilot 运行慢、数值异常、存储膨胀 | 20-event gate；网格敏感性；失败留痕；go/no-go 后才扩 180 |
| V2-R09 | 阶段、风险、趋势或并列阈值不稳定 | 高/高 | 小扰动引起标签翻转 | 明确 TBD；pilot 敏感性；领域复核；边界样本 ambiguous |
| V2-R10 | S/I/V 为追求覆盖而信息不足 | 中/高 | builder 为所有任务强制三轨 | source/task/track eligibility；unsupported 明报而非造题 |
| V2-R11 | 正确选项位置或表述偏差 | 中/中 | A/B/C/D 分布差 >2% 或长度明显不同 | 全局约束采样、长度/风格审计、25% chance sanity |
| V2-R12 | 许可或再发布范围不完整 | 高/高 | 外部源无官方条款/精确版本 | 接入前许可门禁；unknown 不进正式计分/发布 |
| V2-R13 | FireState Card 扩张为复杂方法 | 中/中 | 加入 Agent、训练、检索或 4D 重建 | 只允许五类确定性字段与配对 proof-of-concept |
| V2-R14 | 旧全量测试阻碍新定义或造成假兼容 | 高/中 | 为通过旧测试修改九任务语义 | 新 `test_v2_*` 为当前契约；旧测试只审计 legacy，不反向约束 v2 |
