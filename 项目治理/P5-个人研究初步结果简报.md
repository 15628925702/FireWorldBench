# FireWorldBench 个人研究初步结果简报

## 目前已经有什么

- D01 官方 192 个 CSV 已全部下载并逐文件验证，合计 1.704 GB，无大小或 Git blob SHA 不一致。
- 以 `M/U + case number` 为 group，将 70/100/130 三个位置固定放入同一 partition，避免同一物理组跨集合。
- 普通可见拆分：train 3,300 samples，dev 750，holdout 750；group overlap 为 0。
- 九个任务均有样本；T3-B 以 70m vs 130m 的单变量位置 pair 构建。
- 所有结果是 D01 自动规则标签下的个人研究初步结果，不是论文正式结论。

## 可以怎么向老师解释

当前已经不是四个 case 的 smoke test，而是覆盖 D01 全部 192 个 case 的可复现研究链。简单 majority 在 T1/T2 上明显超过 chance，但在趋势、反事实 pair 和状态 trace 上表现弱，尤其 T3-C 为 0。这说明现有数据和任务能暴露“简单分类规则有效、演化推理仍失败”的差异，后续可以围绕 temporal/causal reasoning 做方法研究。
