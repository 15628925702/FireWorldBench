## 目前已经完成什么

- D01 官方 192 个 CSV 已全部下载并逐文件校验，总计 1.704 GB，文件大小与上游 Git blob SHA 全部匹配。
- 已按 `M/U + case number` 做 group-first 可见划分，train/dev/holdout 分别为 3,300 / 750 / 750，且 group overlap 为 0。
- 九个任务的零成本 baseline 已全部跑通并完成评分。
- DeepSeek 18 条平衡代表样本已经实际运行并评分完成；每个任务 2 条，不扩到 750 条付费运行。
- 当前所有结果都属于 D01 自动规则标签下的 preliminary personal research，不是正式 benchmark，也不是论文结论。

## 可以怎么向老师解释

现在我们已经不只是 4 个 case 的 smoke test，而是具备了覆盖 D01 全部 192 个 case 的可复现个人研究链路。零成本 baseline 表明：

- T1/T2 这类较直接的分类与状态识别任务已经有明显可用信号；
- T3-A/T3-B/T3-C 是最清楚的方法研究空间，尤其 T3-C 在 baseline 下为 0；
- DeepSeek 在 18 条平衡代表样本上的结果非常强，说明当前 prompt + scorer 链路是可跑通的，但这仍然只是小样本研究信号，不是正式论文结论。

## 仍然缺什么

- 自动规则标签还没有专家审核。
- 当前只覆盖 D01，还没有扩展到更完整的数据来源组合。
- 数据许可和正式 benchmark 资格还没有清理完成，所以这些结果只能用于个人研究和导师讨论，不能直接对外宣称为论文结果或正式 benchmark。
