# 项目治理索引

本目录保存跨阶段、跨会话都需要追踪的项目事实：

- `决策记录.md`：已提出/接受/拒绝/替代的架构和研究决策。
- `风险登记.md`：概率、影响、触发器、缓解和责任。
- `开放问题.md`：必须由证据或用户/导师决定的未决事项。
- `需求追踪矩阵.md`：核心 PDF/章程需求到设计、实现和验证的映射。
- `数据资产登记.md`：每个来源的本地状态、角色、资格和下一门禁。
- `论文证据矩阵.md`：预期主张到实验、统计和限制的映射。
- `术语表.md`：项目内统一语义，避免同名异义。
- `测试集访问记录.md`：P2 激活后的 test embargo 访问/拒绝/污染 ledger。

规则：决策记录追加而不删除；风险和开放问题可更新状态但保留历史日期；任何 `UNKNOWN` 不得改写成已经完成。
